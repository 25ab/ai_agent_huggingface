# 🧠 Agentic AI Web Research Assistant

An autonomous AI agent built using **smolagents** and **Phi-3-mini** that can:
- Search the web
- Visit webpages
- Extract content
- Reason step-by-step
- Execute tool-based actions

---


### Agent Workflow

1. User asks a question
2. Agent plans next step
3. Calls tool (web_search / visit_webpage)
4. Receives observation
5. Continues reasoning(reflexion design pattern)
6. Returns final answer

---

## 🛠 Tech Stack

- Python
- smolagents (CodeAgent)
- Phi-3-mini-4k-instruct
- DuckDuckGo Search (ddgs)
- Requests
- Markdownify
